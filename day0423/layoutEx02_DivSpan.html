=> 줄바꿈이 있는 block
=> 콘텐츠의 영역이나 그룹화를 할때 사용
=> 배경색은 레이아웃 가로 세로 모두에 적용
=> 외관 관련 속성은 css(style) 만 적용 됨.
외관 관련 속성 없음

** 2. SPAN **


=> 줄바꿈이 없는 inLine 영역
=> 다른 텍스트와 구별하기 위해 사용
=> 배경색은 글자가 있는곳만 적용
=> 외관 관련 속성은 css(style) 만 적용 됨.
