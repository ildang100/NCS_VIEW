** positioning 관련 속성
=> absolute : 다른 태그 무시하고  무조건 위치하므로 다른 태그들이 가려짐  
=> 해결1: 부모태그에 크기를 지정해서 영역을 만들어줌. 
=> 해결2: 부모의 position : relative 로 
=> 해결3: 부모의 overflow : hidden/scroll 으로

** overflow : 내부의 요소가 부모의 범위를 벗어날때 요소를 처리하는 방법 
=> hidden, scroll, ..

** z-index : 요소의 화면표시 순서를 설정
=> 숫자 값이 클수록 우선순위가 높고 앞쪽에 나타남.
